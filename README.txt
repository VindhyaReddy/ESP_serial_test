ESP_serial_test:

This project is to test ESP8266 module communication with any microcontroller.
Establish UART Communication.
Connect Rx --> Tx
        Tx --> Rx
        GND --> GND
        RST --> one of GPIO pin
Send all AT commands on USB console for checking module responses to AT command set. 
Send AT commands supporting ESP8266 as mentioned in below link:
https://cdn.sparkfun.com/assets/learn_tutorials/4/0/3/4A-ESP8266__AT_Instruction_Set__EN_v0.30.pdf
